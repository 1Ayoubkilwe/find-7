## Find smallest and biggest

##### Your Task 📚:

Create a function called `minMax` that takes an array of numbers and return both the minimum and maximum numbers, in that order.

##### Examples 😃:

```
minMax([1, 2, 3, 4, 5]) ➞ [1, 5]
minMax([300, 5]) ➞ [5, 300]
minMax([1]) ➞ [1, 1]
```

Good Luck 😀
